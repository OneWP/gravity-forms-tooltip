With this plugin you can attach tooltips to any field in gravity forms.

== Description ==
Need to add tooltips to your forms to better enhance communication and messaging on your Gravity Forms?
Easy Gravity Tooltip makes it really easy to do just that!

**How to use**
1. First build a form (using GRAVITY FORMS)
2. Add a text or HTML in "Appearance/Tooltip Content"
3. Save the from and you are done
4. Now you can see the tooltip in field label hover at frontend

**Notice** 
If you want to change appearance of tooltip you can go to "the form/settings/easy tooltip" .
Make sure you also have Gravity Forms activated (at least v2.0).

== Installation ==

This section describes how to install the plugin and get it working.
1. Upload `easy-gravity-tooltip.zip ` to the `/wp-content/plugins/` directory
2. Extract zip file and remove that the zip file
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
= Are there any settings I need to configure ?=
Yes, you can find tooltip settings in “Forms / Settings / Easy Tooltip” .
=Can I use HTML in tooltip content ?=
Yes, you can use HTML and inline css code in tooltip content.
